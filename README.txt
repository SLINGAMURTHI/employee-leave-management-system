Employee Leave Management System

Project Description:
This is a web-based Employee Leave Management System developed to automate
the process of applying, approving, and tracking employee leave requests
in a small organization.

Technologies Used:
- HTML
- CSS
- JavaScript
- MySQL (Database Design)

Project Structure:
EmployeeLeaveManagementSystem/
├── css/        (Stylesheets)
├── html/       (HTML pages)
├── js/         (JavaScript logic)
├── database/   (MySQL SQL file)
└── README.txt

How to Run the Project:
1. Open the project folder in VS Code.
2. Open html/login.html.
3. Right-click and select "Open with Live Server".
4. Use demo credentials to log in.

Note:
This project demonstrates frontend implementation and database design.
Backend integration can be added as a future enhancement.
